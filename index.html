O que é Git?

Git é um sistema de controle de versão de software, facilitando o gerenciamento e compartilhamento de versões. Ele é usado por desenvolvedores de software, principalmente nos porjetos em equipe, por permite que outros membros tenham livre acesso as versões de arquivo para trabalharem no mesmo ritmo.

O que GitHub?

O GitHub é uma plataforma aseada em nuvem que utiliza o sistema Git. De forma simplificada, ela tem o mesmo proposito de controle de versionamento, porém, também é utilizada com rede social entre programadores para compartilhamento de codigo e portfolio.

O comando init é usado para iniciar um repositorio no diretorio local, ou seja, no diretorio atual.
Exemplo: git init
O comando commit grava as alterações no repositorio atual. É importante explicar as alterações feitas para controle e organização das versões, assim é possivel saber as alterações sem precisar abrir o arquivo.
Exemplo: git commit -m "mensagem das alterações feitas"
O comando status exibe o status atual do repositorio, exibindo mudanças que ainda não foram confirmadas, por exemplo.
Exemplo: git status
O comando add adicionanda um arquivo a "area de preparação", significa que esta aguardando ser "commitado".
Exemplo: git add index.html
O comando clone copia um repositorio remoto, para sua maquina. Devemos usar o clone sempre que precisar o projeto completo, historio, branches etc, o que é aconselhavel caso va fazer modificações no projeto. Se for so leitura, ai temos o download zip.
Exemplo: git clone https://github.com/ana-bia07/IEC-Git-Github.git
O comando push envia seus commit locais para o reposito rio remoto (GitHub).
Exemplo: git push origin main
origin main é o local dentro do repositorio.
O comando pull baixa as ultimas mmudanças do repositorio remoto e aplica no repositorio local. Serve pra manter seu codigo atualizado com o q esta no GitHub.
git pull origin main
O comando branch cria uma ramificação. Serve para criar um "caminho" e trabalhar em uma mudança especifica.
Exemplo: git branch nome-funcionalidade or git branch #para listar os branches
O comando une branch com o main. Incorpora o que foi feito de volta na branch principal.
Exemplo: git checkout main git merge nova-funcionalidade

Caso tenha interesse, criar uma conta no GitHub é simples e eles oferecem um curso gratis de como usar.
